# Build KPM Anywhere

## 如何开发

1. 设置环境变量NDK_PATH为NDK的根目录的绝对路径
2. 设置临时变量KP_DIR指向KernelPatch目标源码的根目录（绝对或者相对都可以）
3. 执行make即可

## 支持平台
理论上支持大多数平台上编译 (Windows, Linux, MacOS)
